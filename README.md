# 🇹🇭 Bangkok & Pattaya Bro Trip 2026

A clean, editable trip planner website for the squad.

## Features
- 📅 7-day itinerary — Bangkok → Pattaya → Bangkok
- ✏️ Edit mode: modify any day's activities, title, location, and notes
- 🏨 Add hotel and flight details anytime (saved to browser)
- 🗺️ Interactive Google Maps — switch between Bangkok and Pattaya
- 💾 All edits auto-saved in browser localStorage
- 📱 Mobile friendly

## How to use
1. Open the site
2. Click **"Edit Trip"** button on the hero to update dates, names, tagline
3. Click **"Edit"** on any hotel/flight card to fill in your booking details
4. Click **"Edit"** on any day card to update activities
5. Share the link with your bros!

## Deploy on GitHub Pages
1. Go to your repo Settings → Pages
2. Source: Deploy from branch → `main` → `/ (root)`
3. Save — your site will be live at `https://JhonTimid.github.io/bro-trip-thailand`

## Local
Just open `index.html` in your browser — no build step needed.
